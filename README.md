# E-Commerce-Sales-Management
SQL and Power BI project for analyzing e-commerce sales using MySQL and interactive dashboard
## 📌 Project Overview

This project demonstrates how SQL and Power BI can be used together to analyze and visualize e-commerce sales data.

The database was designed in MySQL, populated with realistic data, and connected to Power BI to create an interactive sales dashboard.

---

## 🚀 Technologies Used

- 🛢️ MySQL 8.0
- 💻 MySQL Workbench
- 📊 Microsoft Power BI
- 📁 CSV Dataset
